## Getting Started

### Installation

1. Clone the repository in the current working directore:
  
    ```bash
    git clone git@github.com:T1mC0de/personal_blog.git
    cd personal_blog
    ```

2. Install dependencies:
  
    ```bash
    pip install -r requirements.txt
    ```

3. Start the development server:
  
    ```bash
    python3 main.py
    ```
    or
    ```bash
    python main.py
    ```
